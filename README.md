# open_lark
OpenLark: Lark Open Platform Python SDK  / 飞书开发平台 Python SDK

从公网项目 https://github.com/larksuite/feishu Fork 而来

## 使用方法

```python
from feishu.api import OpenLark

lark = OpenLark(app_id, app_secret)
```
### 多维表格

```
# api_bitable.py
lark.xxxx
```
